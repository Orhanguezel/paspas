# E3 Adim 3 - Kapanis/Teslim Ozeti Mesaji

P0/P1 fix turu bittikten sonra asagidaki mesaji kullan:

```text
E3 fix turu kapanis ozetini hazirla.

Referans dosyalar:
- docs/E3_TRIAGE_REPORT_READY.md
- docs/E3_FIX_QUEUE.md
- docs/E3_SMOKE_TEST_COMMANDS.md
- docs/PILOT_DELIVERY_PACKAGE_CHECKLIST.md
- docs/E3_OPERATION_BOARD.md

Yapilacaklar:
1) Kapatilan buglari (ID + oncelik + etki) listele.
2) Acik kalan buglari ayri listele:
   - blocked-by-api
   - sonraki sprint (P2/P3)
3) Son smoke/build sonucunu ozetle:
   - backend build
   - admin_panel build
4) E3 maddeleri icin final durum ver:
   - PASS
   - FAIL
   - BLOCKED
5) Pilot teslim paketinde hazir olanlari ve eksikleri isaretle.

Beklenen cikti:
- Tek sayfada kapanis ozeti
- Kalan riskler
- Bir sonraki aksiyon listesi (net 3-5 madde)
```
